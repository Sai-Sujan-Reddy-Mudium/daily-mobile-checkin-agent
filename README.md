# Daily Mobile Check-In Agent

## Problem
Checking WhatsApp messages and Gmail updates every day requires repetitive manual actions.

## Solution
This project uses the Droidrun framework to automate a daily mobile check-in workflow.
The agent navigates through WhatsApp and Gmail on an Android device and performs basic interactions
such as opening chats, scrolling, and checking inbox updates.

## Workflow
1. Open WhatsApp
2. Navigate to a predefined chat
3. Scroll and pause
4. Open Gmail
5. Open inbox and scroll

## Tech Stack
- Droidrun framework
- Android Emulator / Android Device

## Status
Initial prototype under development.
